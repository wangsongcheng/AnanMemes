## 夏目安安表情包制作器
```bash
usage:./a.out 文本 [-i图片名称] [-f表情索引] [-o <图片名称>]
	表情索引:
		0 = 默认
		1 = 高兴
		2 = 病娇
		3 = 脸红
		4 = 愤怒
		5 = 无语
注意:
	换行符是空格
	[]必须用英文符号
	输出的文件只支持jpg、png图片
	有概率会发生文字重叠, 可以通过多次生成来重新调整位置
	如果多次调整仍有文字重叠, 则建议减少字数
```
